Instrucciones de la Práctica

En esta práctica veremos el ciclo completo de DevOps. Realizar lo siguiente:

1

Desarrollar una Aplicación Web

Realizar una app web "Hola Mundo" en tu lenguaje de programación favorito.

2

Crear un Dockerfile

Utilizar Docker para crear un Dockerfile y generar una imagen de tu aplicación.

3

Publicar en Docker Hub

Subir la imagen de tu aplicación a Docker Hub para compartirla.

Ejemplo de Dockerfile

A continuación, un ejemplo básico de Dockerfile para una aplicación web simple:

# Usar una imagen base oficial de Node.js FROM node:14-alpine # Establecer el directorio de trabajo WORKDIR /app # Copiar los archivos de la aplicación COPY package*.json ./ RUN npm install # Copiar el resto de los archivos COPY . . # Exponer el puerto que usa la aplicación EXPOSE 3000 # Comando para ejecutar la aplicación CMD ["npm", "start"]
